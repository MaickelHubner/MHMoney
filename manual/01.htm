2. Como instalar o MH Money 2005

 

Para a completa instalação do MH Money 2005, siga os passos abaixo:

1. Dê duplo-clique com o mouse sobre o arquivo de instalação (.exe).
2. Abrirá uma tela de instalação como a mostrada abaixo:

3. Informe um diretório onde serão baixados os programas em "Extract to" (como sugestão C:\MHMoney).
4. Clique no botão Start.

Os arquivos do MH Money 2005 serão então descompactados no diretório informado. Ao final do processo será mostrada uma mensagem como a abaixo:

Depois é necessário acessar o diretório C:\MHMoney e executar o ícone . Isso iniciará o programa de instalação do MH Money 2005, abrindo a tela abaixo:

Nesta tela é possível marcar três opções, explicadas abaixo:

Criar Base de Dados Nova - Essa opção, caso marcada, irá gerar uma base de dados nova. Se você estiver atualizando sua versão do MH Money 1.1, não deverá marcá-la.

Criar as principais Categorias\Sub-Categorias - Essa opção só será habilitada caso a opção Criar Base de Dados Nova estiver selecionada. Categorias e Sub-Categorias são a forma como o MH Money 2005 divide ou agrupa as suas despesas. Se esta opção for selecionada o MH Money 2005 irá criar uma série de categorias e sub-categorias pré-definidas. Futuramente voce poderá eliminá-las ou alterá-las, além de incluir suas próprias categorias e sub-categorias. Se estiver em dúvida quanto a essa opção, sugerimos deixá-la marcada.

Cadastrar a moeda principal - Diferente do MH Money 1.1, o MH Money 2005 trabalha com o conceito de moedas. Você poderá visualizar e controlar suas contas em qualquer moeda (Dólar, Euro, etc). Caso essa opção for marcada, o MH Money 2005 irá assumir a moeda principal como sendo Reais (R$) e irá efetuar as configurações necessárias. Se essa opção for desmarcada você deverá cadastrar manualmente a moeda principal do sistema depois.

Quando terminar suas escolhas, clique no botão Continuar... . Será aberta a tela abaixo:

Quando o sistema terminar de gerar a nova base de dados, será habilitado o botão Continuar... . Ao clicar nele, será mostrada a mensagem abaixo:

Essa parte da instalação é muito importante, serão mostradas as informações abaixo na tela principal do programa:

Ao fundo será aberto o Data Administration, um programa de configuração do banco de dados.

É necessário seguir os passos abaixo:

1. No menu, selecione Admin \ Load Data and Definitions \ Data Definitions (.df file)...

2. Abrirá a tela abaixo:

3. Clicar no botão OK.

Ao final do processo será mostrada a mensagem abaixo:

Nesse ponto você pode fechar o programa, e na tela de instalação clicar no botão Continuar... . Será mostrada então a tela abaixo:

O MH Money 2005 irá então criar as Categorias e Sub-Categorias e configurar a moeda principal, caso essa opções estiverem marcadas na janela inicial. Além de realizar diversas configurações adicionais. Depois você deve clicar no botão Continuar... . Será mostrada então a tela abaixo:

Nesta tela será solicitado o usuário e senha principais do sistema. É com esse usuário e essa senha que você poderá acessar o MH Money 2005 pela primeira vez após a instalação. É importante não informar uma senha em branco, pois futuramente não será possível acessar o MH Money 2005 nessa situação. Após informar, clique no botão Continuar... . Será mostrada a mensagem abaixo:

Isso indica que o processo de instalação terminou. Será aberta a tela de login abaixo automaticamente:

É necessário informar um usuário cadastrado no sistema e sua senha. No primeiro acesso deverá ser informado o usuário criado durante a instalação. Futuramente você poderá cadastrar mais usuários para acessarem o sistema, ou até mesmo alterar a senha de usuários existentes.

Após informar o usuário e senha, o botão Efetuar Login... será habilitado. Clique nele para acessar a tela principal do MH Money 2005.

Para acessar a tela de Login novamente, você deverá clicar no ícone que se encontra no diretório de instalação (C:\MHMoney).
